# Portable by default — no machine-specific absolute paths live here:
#   • ${HYPEPROOF_WORKSPACE} is the workspace base. If you don't set it,
#     sync.sh / tests/run.sh default it to the PARENT of this repo, so
#     cloning the consumers as siblings of hypeproof-harness just works
#     with zero config:
#         <workspace>/hypeproof-harness   (this repo)
#         <workspace>/hypeproof-studio
#         <workspace>/sediment
#   • To use a different base on a given machine:
#         export HYPEPROOF_WORKSPACE=/abs/path/to/workspace
#   • To point a single repo somewhere unrelated, override per-repo
#     (dashes in the basename become underscores in the env var name):
#         CONSUMER_hypeproof_studio=/abs/path/to/clone bash scripts/sync.sh
#
